# Caspeco Frontend Demo

This is a demo project showcasing Caspeco's restaurant management platform — a POS and analytics solution for the restaurant industry.

Caspeco provides tools for point-of-sale, powerful analytics, purchase optimization, and financial reporting for restaurants.

## Project structure

- `research/` — Contains research data, screenshots, and reference material gathered from Caspeco's website and product.
- `spec/` — Contains the specifications for features and components that should be built.

## Active Technologies
- HTML5 + CSS3 + ES6+ JavaScript (Phase 1), TypeScript + Next.js 14 (Phase 2) + Chart.js via CDN (Phase 1); Next.js, Tailwind CSS, shadcn/ui, html2canvas (Phase 2) (001-caspeco-restaurant-demo)
- N/A — all data is hardcoded in-memory (001-caspeco-restaurant-demo)
